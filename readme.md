Generador aleatorio de CUPS

## Usage

```sh
docker build --no-cache -t "cups-generator" ./
docker run -v "`pwd`:/code" cups-generator composer install
docker run -v "`pwd`:/code" cups-generator vendor/bin/phpunit
```
